Set up python2.7

To set up python2.7 on MacOs

  • Follow https://stackoverflow.com/questions/67380286/anaconda-channel-for-installing-python-2-7

  • Install additional packages with Env/py27.yml

Then run

conda create -n py27 --f py27.yml

To active it

conda activate py27;
